# Sport Facility Booking System — Backend

FastAPI (Python) REST API for the Sport Facility Booking System.

## Tech Stack
- FastAPI + Python 3.11
- PostgreSQL + SQLAlchemy + Alembic
- RabbitMQ (async notifications & payments)
- Stripe (payment gateway)
- OAuth2 (Google login)
- pytest (unit + integration tests)
